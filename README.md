# NAS System Monitor / NAS 系统监控

## 功能简介 | Features
- 复古桌面式窗口界面，支持拖拽与缩放，可在桌面和移动端顺畅操作。
- 实时展示 CPU、内存、网络、存储、Docker 等关键指标，并包含滚动式列表与进度条。
- 提供布局保存与恢复功能，方便按个人习惯整理监控面板。

## 部署说明 | Deployment
- 请前往 Release 页面获取最新构建与部署指引。
- Refer to the Release page for the latest build artifacts and deployment steps.

## 备注 | Notes
- 页面样式基于 Tailwind CSS 与自定义像素风格，兼顾现代浏览器与触控设备。
- 数据接口由 `server` 目录中的 Go 程序提供，可按需扩展。
